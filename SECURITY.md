# Security Policy

## Supported Versions

The following versions are currently being supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| v0.31.x   | :white_check_mark: |
| v.0.30.x   | :white_check_mark: |
| < v0.30.0   | :x:                |

## Reporting a Vulnerability

To report a vulnerability, please submit an advisory [here](https://github.com/FlexMeasures/flexmeasures/security/advisories).

We will get back to you within a week. If we accept the advisory, you will be credited as reporter.
